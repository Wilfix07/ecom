## 📝 Utilisation

### Pour Uploader:
1. Cliquez sur la zone pointillée
2. Sélectionnez une image
3. Attendez le spinner
4. ✅ Image uploadée!

### Pour Remplacer:
1. Survolez l'image existante
2. Cliquez sur le bouton [+] bleu
3. Sélectionnez une nouvelle image
4. ✅ Image remplacée!

### Pour Supprimer:
1. Survolez l'image
2. Cliquez sur le bouton [X] rouge
3. ✅ Image supprimée!
